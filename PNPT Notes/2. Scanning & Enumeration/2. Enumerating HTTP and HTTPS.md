# Enumerating HTTP/HTTPS

When finding ports like 80/443, go to the webpage to see what is on it.

## Default Webpages
Default webpages are technically a finding. It says a lot about the potential hygiene of the client.

If they are running a default webpage, what else do they have set up thats default?

Always check Headers in the response of the webpage. Server headers sometimes disclose version information

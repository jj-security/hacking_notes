### Use netdiscover with range to use ARP to detect all machines on the network.

``` 
netdiscover -r 192.168.1.0/24
```

# Nmap
## Flags

- `-p 22,23,24`: Scan a single port or a comma separated list
- `--top-ports <number>`: Scan top X ports. Top 100 or 1000 usually.
- `-p-`: Scan all ports
- `-A`: Enable OS detection, version detection, script scanning, and . Is slow, because of version detection and scripts
- `-T<1-4>`: Timing options, 1 is slowest, 4 is fastest
- `-sn 192.168.1.0/24`: ping sweep a subnet
- `-sT`: Full TCP handshake
- `-sU`: UDP scan. Very slow, should use with `--top-ports 1000` flag or smaller
- `-sS`: Half open scan, syn/syn ack/rst, a.k.a: Stealth Scan
- `-oN <file_name_and_path>`: output to a file using nmap format
- `-oG <file_name_and_path>`: output to a greppable file format
- `-oA <file_name_and_path>`: output to all file formats
- `-sV`: Once port scan is complete, get all service version names if you can.
